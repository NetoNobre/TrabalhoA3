# Introdução à Bifrost

Vamos descobrir como inicializar o  **Bifrost em menos de 10 minutos**.

## Comece criando uma nova pasta.

Nessa pasta você irá **clonar um repositório**.

**Entre no repositório e baixe a pasta em zip** with **[TrabalhoA3](https://github.com/NetoNobre/TrabalhoA3)**.

### O que você irá precisar

- [Node.js](https://nodejs.org/en/download/) version 16.14 or above:
  - When installing Node.js, you are recommended to check all checkboxes related to dependencies.

## Instalando dependências

Instale as dependências do Node.JS  **com o comando abaixo**.

O comando abaixo irá instalar as dependências na sua pasta raiz:

```bash
npm install
```

Você pode digitar este comando no prompt de comando, Powershell, Terminal ou qualquer outro terminal integrado do seu editor de código.



## Comece a navegar 

Execute o servidor de desenvolvimento:

```bash
cd trabalhoa3a
npm run dev
```

O `cd` comando altera o diretório com o qual você está trabalhando. Para navegar na sua plataforma recém-clonada, você precisará navegar no terminal até lá.

O `npm run dev` comando constrói seu site localmente e o veicula através de um servidor de desenvolvimento, pronto para você visualizar em http://localhost:3000/.

### Instalação do React

Para caso vocês (dev's) queiram criar um projeto do zero seguindo o mesmo modelo, por exemplo, vocês precisarão instalar o React. Segue abaixo o passo a passo de como instalar o React:

Diante mão, é necessário ter instalado o Node. Portanto, como foi feito a inicialização da plataforma Bifrost, provavelmente vocês já têm instalado, caso não, o link para download está no tópico "O que você irá precisar".

- Crie uma pasta onde será inicializado o seu projeto em React
- Feito isso, dentro da pasta criada, clique com o botão direito do mouse e clique em "Abrir no Terminal"
- Em seguida, já dentro do terminal, execucte o comando:

```
npx create-react-app (o nome do seu projeto)
```
- E então, criado o seu projeto em React, digite os comandos a seguir para executá-lo:

```
cd (o nome do seu projeto)
npm start
```
