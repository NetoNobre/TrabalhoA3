# Introdução à Bifrost

A Bifrost é de uma plataforma que gerencia e apresenta as principais ferramentas de engenharia de software. O objetivo do  da plataforma é ser um ambiente que possa reunir as principais ferramentas que poderão auxiliar qualquer desenvolvedor no processo de criação dos modelos de processos de software, na análise, especificação, validação e gestão de requisitos, ferramentas para testes de software, prototipação, gerenciamento de projetos,  ferramentas base para framework ágil e qualquer outra ferramenta que seja útil para o desenvolvimento de qualquer aplicação.

# Filtro de pesquisa na prática

Na imagem abaixo é mostrado o funcionamento do filtro de pesquisa na prática. O filtro de pesquisa, foi pensado em uma forma mais fácil de encontrar a sua ferramenta. Ele foi pensado também em encontrar a sua ferramenta independente se for pesquisado o nome com letra minúscula ou maiúscula. Como é mostrado, foi digitado um "v", então ele automaticamente lista todas as ferramentas que contêm a letra "v" no nome. É um filtr ode busca inteligente e o principal método responsável por isso é o LowCase.

# Pesquisa por categoria

Na imagem abaixo, é mostrado o funcionamento da pesquisa por categoria, a fim de facilitar também a usabilidade do usuário. Quando é selecionado a categoria da ferramenta de software, ele automaticamente lista um Array com todas as ferramentas de software da categoria selecionada. Isto foi projetado justamente para facilitar a busca de ferramentas, selecionando por categoria.

![filtropesquisa](/img/pesquisacategoria.jpg)


# Limpa filtro de pesquisa

Na imagem abaixo nota-se que, o botão "LIMPAR" ao ser clicado, ele retorna para listagem inicial das ferramentas de software. Isso acontece porque a função foi projetada para apagar o Array atual que é criado quando pesquisa ou seleciona uma categoria, pelo Array inicial. Esta função foi projetada para facilitar a usabilidade do usuário que não precisará recarregar a página para mostrar todas as ferramentas novamente.

Vamos descobrir como inicializar o  **Bifrost em menos de 10 minutos**.

## Comece criando uma nova pasta.

Nessa pasta você irá **clonar um repositório**.

**Entre no repositório e baixe a pasta em zip** with **[TrabalhoA3](https://github.com/NetoNobre/TrabalhoA3)**.

### O que você irá precisar:

- [Node.js](https://nodejs.org/en/download/) version 16.14 or above:
  - When installing Node.js, you are recommended to check all checkboxes related to dependencies.

## Instalando dependências

Instale as dependências do Node.JS  **com o comando abaixo**.

O comando abaixo irá instalar as dependências na sua pasta raiz:

```bash
npm install
```

Você pode digitar este comando no prompt de comando, Powershell, Terminal ou qualquer outro terminal integrado do seu editor de código.



## Comece a navegar 

Execute o servidor de desenvolvimento:

```bash
cd trabalhoa3a
npm run dev
```

O `cd` comando altera o diretório com o qual você está trabalhando. Para navegar na sua plataforma recém-clonada, você precisará navegar no terminal até lá.

O `npm run dev` comando constrói seu site localmente e o veicula através de um servidor de desenvolvimento, pronto para você visualizar em http://localhost:3000/.

### Instalação do React

Para caso vocês (dev's) queiram criar um projeto do zero seguindo o mesmo modelo, por exemplo, vocês precisarão instalar o React. Segue abaixo o passo a passo de como instalar o React:

Diante mão, é necessário ter instalado o Node. Portanto, como foi feito a inicialização da plataforma Bifrost, provavelmente vocês já têm instalado, caso não, o link para download está no tópico "O que você irá precisar".

- Crie uma pasta onde será inicializado o seu projeto em React
- Feito isso, dentro da pasta criada, clique com o botão direito do mouse e clique em "Abrir no Terminal"
- Em seguida, já dentro do terminal, execucte o comando:

```
npx create-react-app (o nome do seu projeto)
```
- E então, criado o seu projeto em React, digite os comandos a seguir para executá-lo:

```
cd (o nome do seu projeto)
npm start
```
