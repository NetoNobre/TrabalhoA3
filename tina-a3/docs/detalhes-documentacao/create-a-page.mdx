# Tecnologias e Arquitetura

## Objetivo:


A Bifrost é de uma plataforma que gerencia e apresenta as principais ferramentas de engenharia de software. O objetivo do  da plataforma é ser um ambiente que possa reunir as principais ferramentas que poderão auxiliar qualquer desenvolvedor no processo de criação dos modelos de processos de software, na análise, especificação, validação e gestão de requisitos, ferramentas para testes de software, prototipação, gerenciamento de projetos,  ferramentas base para framework ágil e qualquer outra ferramenta que seja útil para o desenvolvimento de qualqueraplicação.


## Ambiente de Desenvolvimento:
▪️ Linguagens de Programação: HTML, CSS, JAVASCRIPT

▪️ IDE: VSCode

▪️ Frameworks: React e Node.js
